# Contributing to Fluxer

First off, thank you for considering contributing to Fluxer! It's people like you who make this a better tool for everyone.

## Code of Conduct

By participating in this project, you agree to abide by our code of conduct: be respectful, professional, and helpful.

## How Can I Contribute?

### Reporting Bugs
- Use the GitHub Issue Tracker.
- Describe the bug and provide steps to reproduce it.
- Include your device model and Android version.

### Suggesting Enhancements
- Open a "Feature Request" in the issue tracker.
- Explain why the feature is needed and how it benefits users.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. Ensure your code follows the existing style and is linted.
3. Submit the PR with a clear description of the changes.
