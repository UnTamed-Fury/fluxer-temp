# Security Policy

## Supported Versions

Currently, only the latest version of Fluxer Android Wrapper is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of this project seriously. If you believe you have found a security vulnerability, please do NOT open a public issue.

Instead, please report it privately by contacting **UnTamed-Fury** via GitHub or the official Fluxer community channels.

Once a report is received, we will:
1. Acknowledge the report within 48 hours.
2. Provide an estimated timeline for a fix.
3. Notify the community once a patch is available.

Thank you for helping keep this project safe!
